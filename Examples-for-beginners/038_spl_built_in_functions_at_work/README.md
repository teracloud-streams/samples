## Overview
This is a very simple example that showcases a random collection of powerful built-in SPL functions that are availble out of the box. This application demonstrates how time, math, and collection type functions can be used inside of an SPL application.

For more information about how to compile, execute and view the output of a sample application, see the [Build, Run, View Guide](../../BuildRunView.md)
